Arquitectura

Bitnativo es mi primer proyecto web, trabajé en el de manera integral desde la idea de negocio hasta el backend. La base de su arquitectura se encuentra montada desde cero (on bare), en un servidor ubuntu 16.04 LTE, sin CMS. Nginx es inicialmente el encargado de resolver la petición web realizada por el cliente, re-enviandola para que Gunicorn pueda procesarla utilizando varios workers. Cada worker es capaz de comunicarse con la aplicación Django de manera que pueden atenderse multiples peticiones al mismo tiempo, manteniendo un buen tiempo de respuesta. Por otro lado, Supervisor se encargará de controlar la ejecución del proceso Gunicorn de tal forma que podamos iniciar, detener o reiniciar la aplicación de Django en el momento que sea necesario. Como motor de base de datos utilizamos Postgresql.

Seguridad

Como arquitectos y desarrolladores de software, hoy en día tenemos la obligación de hacer una web rápida y segura.
SSL / TLS es una tecnología aparentemente simple. Es fácil de implementar y simplemente funciona. . . En realidad, esto no suele ser así. La primera parte es verdadera: SSL es fácil de implementar, pero resulta que no es fácil de implementar correctamente. Para garantizar que SSL proporcione la seguridad necesaria, los usuarios deben esforzarse más en configurar correctamente sus servidores.
En este contexto, el objetivo en bitnativo era obtener la máxima puntuación posible en el test de SSLLabs cumpliendo los últimos estándares mientras mantenemos máxima compatibilidad.

Velocidad

Puntuación de velocidad
En PageSpeed Insights (PSI) se proporcionan datos sobre el rendimiento real de las páginas tanto en dispositivos móviles como en ordenadores. Para evaluar la carga de nuestra página web así como nuestra implementación, PageSpeed Insights se basa en un documento de buenas prácticas que abarca tanto a nivel de front-end (carga de imágenes, archivos JavaScript, CSS, etc) como de configuración de servidor.
bitnativo.com es una pagina rápida, se encuentra en el tercio más rápido de todas las cargas de páginas que analiza PSI.
